Method. 2,835 cleaned UK postings collected from the Adzuna API (June 2026). Titles embedded with all-MiniLM-L6-v2, reduced with UMAP, clustered with HDBSCAN (17 archetypes + 12.6% noise). Salary midpoints blend employer-posted (36%) and Adzuna-predicted (64%) figures.

Caveats. Single-day snapshot; clusters are unsupervised (not validated against a taxonomy); the 463-posting Finance/Consulting cluster is a catch-all. A separate leak-free XGBoost model explains 50% of real-salary variance (MAE ~£17K) — useful for understanding drivers, not individual offers. Built by Jumma Mohammad Teli.
