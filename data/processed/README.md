# Processed Data

This folder contains intermediate and output files generated by the analysis notebooks.

## Files committed to Git (small, useful for inspection)

| File | What it is | Size |
|---|---|---|
| `sample_postings.csv` | 200-row stratified sample of the cleaned dataset, for schema inspection | ~70 KB |
| `model_results_real.csv` | 5-fold CV results for all 4 models on real (employer-posted) salaries | <2 KB |
| `model_results_predicted.csv` | 5-fold CV results for all 4 models on Adzuna's predicted salaries | <2 KB |
| `feature_importance_real.csv` | XGBoost feature importance ranking, real salaries (267 features) | ~15 KB |
| `feature_importance_predicted.csv` | XGBoost feature importance ranking, Adzuna predictions (267 features) | ~15 KB |
